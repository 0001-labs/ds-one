---
title: Gap
---

`<ds-gap>` is a full-width vertical spacer.

## Examples

```html
<ds-gap unit></ds-gap>
<ds-gap double></ds-gap>
<ds-gap size="05"></ds-gap>
```

## Sizes

- Semantic boolean attributes (from `DS1/1-root/one.css`): `tenth`, `quarter`, `half`, `eight-tenth`, `unit`, `double`, `triple`, `quad`, `oct`, `dozen`
- Raw scale via `size="..."`: `01`, `025`, `05`, `08`, `1`, `2`, `3`, `4`, `8`, `12`
